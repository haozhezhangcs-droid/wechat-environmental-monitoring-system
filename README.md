# wechat-environmental-monitoring-system
A WeChat Mini Program for environmental monitoring with temperature, humidity and gas concentration visualization.

## Features

- Temperature monitoring
- Humidity monitoring
- Gas concentration monitoring
- Configurable alarm thresholds
- Interactive user interface

## Technologies

- JavaScript
- WXML
- WXSS
- WeChat Mini Program

## Overview

This project demonstrates a WeChat Mini Program that provides an interactive interface for monitoring environmental parameters and configuring alarm thresholds.
